# 🏥 Overview

**Aarogyavan** is a lightweight, browser-based health tracking system designed to help users **record, organize, and review personal health data** in a structured and intuitive way.

Unlike complex healthcare systems, it focuses on:
- clarity over complexity  
- usability over features  
- structure over clutter  

---

# ⚡ Key Highlights

- 📋 Manual health data entry system  
- 📊 Clean and structured data display  
- 🧾 Organized personal record management  
- 💡 Built-in awareness-oriented design  
- 📱 Fully responsive across devices  
- ⚡ Zero dependencies (pure frontend)
